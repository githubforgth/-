## ✨ 功能特性

*本脚本旨在自动化一些重复性的课程操作，例如：*

-   [ ] **自动播放视频**：无需手动点击，自动播放下一个学习任务点。
-   [ ] **倍速播放**：支持自定义视频播放速度（如 2倍速）。
-   [ ] **无缝跳转**：一个任务点完成后自动进入下一个。

## 🛠️ 安装与使用

### 第一步：安装油猴插件（Tampermonkey）

如果你的浏览器还没有安装油猴，请先安装它。

- **Chrome**: 在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) 搜索 Tampermonkey 并安装。
- **Edge**: 在 [Microsoft Edge 加载项](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd) 商店安装。
- **Firefox**: 在 [Firefox 附加组件](https://addons.mozilla.org/zh-CN/firefox/addon/tampermonkey/) 站点安装。

### 第二步：安装脚本

1.  点击本页面右上角的 `Code` 按钮，然后选择 `Download ZIP` 下载本项目的压缩包。
2.  解压压缩包，找到其中的 `.user.js` 文件（例如：`***.user.js`）。
3.  打开 Tampermonkey 的管理面板（点击浏览器右上角的油猴图标，然后选择“管理面板”）。
4.  将 `.user.js` 文件直接拖拽到 Tampermonkey 的管理面板页面，即可完成安装。
5.  **（或者，更简单的方法）** 如果我在 GitHub 上直接发布了发布版本，你可以在 `Releases` 页面找到安装链接，点击后 Tampermonkey 会提示你进行安装。

### 第三步：开始使用

1.  安装完成后，确保脚本处于启用状态（在 Tampermonkey 面板中开关为绿色）。
2.  访问你的课程平台。
3.  进入课程学习页面，脚本会自动运行。

## ⚙️ 配置说明

脚本通常会提供一个配置面板。你可以通过以下方式打开：
- 点击浏览器右上角的油猴插件图标，在脚本名称下方找到“配置面板”选项。
- 或者在课程页面上寻找浮动的控制按钮。


## ❓ 常见问题 (FAQ)

**Q：脚本不生效怎么办？**
**A：** 1. 检查 Tampermonkey 是否已启用。 2. 检查本脚本是否已启用。 3. 刷新课程页面。 4. 检查脚本是否在正确的网址上运行（在 Tampermonkey 编辑器中查看 `@match` 或 `@include` 字段）。

**Q：课程平台更新后脚本失效了？**
**A：** 这很常见。请在本项目的 [Issues](你的GitHub仓库Issues页面链接) 页面反馈，并描述具体情况。

**Q：使用脚本会被发现吗？**
**A：** **无法保证 100% 安全。** 课程平台有各种技术手段可以检测自动化行为。使用脚本即代表你接受了潜在的风险。
